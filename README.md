# Arduino: CH32V003-Blink-Deluxe

CH32V003 開発ボードを使用したLED点滅とブザー制御のプロジェクトです。

## 部材

| 種別 | 型番など | 数量 |
|:-----|:---------|:-----|
| ブレッドボード | 400H | 1個 |
| ジャンパーワイヤ | - | 適量 |
| タクトスイッチ | 2P 6x6mm | 3個 |
| マイコン | CH32V003 開発ボード | 1個 |
| LED | 3mm または 5mm | 3個 |
| 抵抗 | 200Ω | 3個 |
| パッシブブザー | - | 1個 |
| 抵抗 | 100Ω | 1個 |

## 配線

- **LED1**: PC4（200Ω抵抗経由でGND）
- **LED2**: PC3（200Ω抵抗経由でGND）
- **LED3**: PC2（200Ω抵抗経由でGND）
- **ボタン1**: PC5（INPUT_PULLUP）
- **ボタン2**: PC6（INPUT_PULLUP）
- **ボタン3**: PC7（INPUT_PULLUP）
- **ブザー**: PC1（100Ω抵抗経由でGND）

## プログラム開発環境

### 統合開発環境

**インストール**:
1. 公式サイト [https://www.arduino.cc/en/software/#ide](https://www.arduino.cc/en/software/#ide) からダウンロード・インストール

### ボードマネージャ

**インストール**:
1. Arduino IDE の「ファイル」→「環境設定」→「追加のボードマネージャのURL」に以下を追加:
   ```
   https://github.com/openwch/board_manager_files/raw/main/package_ch32v_index.json
   ```
2. ボードマネージャーで「CH32 RISC-V Arduino」を検索し、「インストール」ボタンをクリック

## ビルド・書き込み手順

1. Arduino IDE で `CH32V003-Blink-Deluxe.ino` を開く
2. 「ツール」→「ボード」→「CH32 RISC-V Arduino」→「CH32V003」を選択
3. 「ツール」→「ポート」でWCH-LinkEが接続されているポートを選択
4. 画面上部の右向き矢印ボタン（アップロード）をクリック

## スケッチの動作

### 起動時

- 起動音が鳴ります
- LED1が点滅を開始します

### ボタン操作

各ボタンを押すと、対応するLEDが点灯し、音階が鳴ります。その後、そのLEDが点滅を継続します。

- **ボタン1**: LED1 + C4（ド）
- **ボタン2**: LED2 + D4（レ）
- **ボタン3**: LED3 + E4（ミ）
