# Arduino: CH32V003-DevKit

CH32V003 開発ボードを使用した、コンパクトな電子工作の入門セットです。

## 機能

- **３ボタン入力**: 各ボタンに異なる音階を割り当て
  - ボタン1: ド (262Hz)
  - ボタン2: レ (294Hz)
  - ボタン3: ミ (330Hz)
- **LED表示**: ボタン押下時にLED点灯
- **ブザー出力**: ボタン押下時に対応する音を200ms鳴らす

## プログラム開発環境

### 統合開発環境

**インストール**:
1. 公式サイト [https://www.arduino.cc/en/software/#ide](https://www.arduino.cc/en/software/#ide) からダウンロード・インストール

### ボードマネージャ

**インストール**:
1. Arduino IDE の「ファイル」→「環境設定」→「追加のボードマネージャのURL」に以下を追加:
   ```
   https://github.com/openwch/board_manager_files/raw/main/package_ch32v_index.json
   ```
2. ボードマネージャーで「CH32V」を検索・インストール

## 操作

USB 端子からマイコンに電源を供給すると作動します。

ボタンを押すと LED が点灯し、対応する音階が 200ms 鳴ります。
